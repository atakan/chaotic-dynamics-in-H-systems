## Dankowicz's book


### Illustration on pp 34-35

H = 1/2*(p1^2+q1^2) + 1/2*(p2^2+2*q2^2) + q1^2*q2
i.e., H = T + V
T = 1/2*(p1^2+p2^2)
V = 1/2*(q1^2+2*q2^2) +  q1^2*q2

what is plotted is the intersections with the q2=0 plane, when p2>0 and H=0.62
